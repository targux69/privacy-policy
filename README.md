## 🛠️ Tecnologías de Datos
Para garantizar que Alle sea multidispositivo y seguro, utilizamos:
* **Base de Datos:** MongoDB Atlas (Región: París).
* **Multimedia (Fotos/Videos):** Cloudinary CDN.
* **Seguridad:** Acceso local mediante PIN de 4 dígitos por perfil.
